# AGENTS.md

This guide helps agentic coding assistants work effectively in this codebase.

## Project Structure

This is a learning repository with multiple technology topics:

- `LLD/` - Low Level Design and OOPS concepts
- `ElasticSearch/` - Elasticsearch tutorials
- `Go/` - Go language projects
- `Java/` - Java basics
- `Node.js/` - Node.js server examples
- `RabbitMQ/` - Message queue examples

## Code Style

- Use meaningful variable/function names
- Add comments for learning purposes
- Keep code simple and readable
